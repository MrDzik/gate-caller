
# GateCaller
.NET MAUI Simple project tergetting Android to handle call-to-open gates.

## Languages
- English
- Polish

## Features

- Light/dark support.
- Automatic calling with only one click.
- Displaying distance to gates.
- Simple and easy to use UI.
